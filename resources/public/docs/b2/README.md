## Project Outline

The project you will learn is to add two numbers. Both the numbers are to be added through two input elements and should output sum after validation on a button click.

## Learning Outcome

* Components
* useState

## Create a Project
```js
npm create vite@latest
```

    - project name : b2
    - framework : react
    - variant : javascript

```js
cd b2
npm install
npm run dev
```
